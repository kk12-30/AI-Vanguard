# AI-Vanguard (先锋者) - 智能自动化安全测试平台

## 📖 简介

**AI-Vanguard (先锋者)** 是一款基于大语言模型（LLM）驱动的新一代自动化渗透测试与信息搜集工具。它不仅仅是一个简单的工具调用器，更是一个具备"思考"能力的智能体（Agent）。通过 ReAct（Reasoning and Acting）范式，它能够理解用户意图，自主制定测试计划，动态调用安全工具，并根据工具反馈结果修正下一步行动，最终生成专业的安全测试报告。

项目致力于降低安全测试的门槛，同时提升专业渗透测试人员的效率，成为您最得力的 AI 安全助手。

## ✨ 核心功能亮点

### 1. 🧠 AI 驱动的智能决策核心

*   **自主规划**：不再依赖死板的脚本流程。AI 根据目标情况实时规划测试路径（例如：发现 80 端口 -> 识别为 Web 服务 -> 启动 Web 扫描 -> 发现参数 -> 测试注入）。
*   **自动纠错**：当工具执行失败或参数错误时，AI 能阅读错误日志并自动修正参数重试，或尝试替代方案。
*   **长链推理**：支持多轮对话迭代，能够处理复杂的逻辑推理任务，一步步深入目标。

### 2. 🎭 多专家角色模式 (Multi-Role System)

内置多种预设的专家角色，针对不同场景提供最专业的测试策略：

*   **✨ 智能模式 (Smart Mode)**：全自动识别用户输入和上下文，智能切换到最适合的专家角色，无需人工干预。
*   **🛡️ 渗透测试专家 (Pentester)**：全方位的综合渗透测试，覆盖从信息搜集到漏洞利用的全流程。
*   **🔍 信息搜集专家 (Recon)**：专注于资产发现、子域名枚举、端口扫描、指纹识别、Whois 查询等前测阶段。
*   **💉 SQL注入专家 (SQLi)**：专注于数据库注入漏洞的探测与验证（深度集成 SQLMap 等工具）。
*   **💻 RCE测试专家 (RCE)**：专注于远程代码执行漏洞的挖掘与利用验证。
*   **🌐 XSS测试专家 (XSS)**：专注于跨站脚本攻击的检测与 Payload 构建。
*   **📂 文件包含专家 (LFI)**：专注于本地/远程文件包含漏洞的探测。
*   **🔑 越权测试专家 (IDOR)**：专注于业务逻辑漏洞和越权访问测试。
*   **🔌 MCP模式**：纯粹的 Model Context Protocol 客户端模式，用于连接外部 MCP 服务，扩展无限可能。

### 3. 🛠️ 强大的工具生态与 MCP 支持

*   **内置/集成主流安全工具**：
    *   **资产发现**：Subfinder, Httpx, Whois
    *   **端口扫描**：Nmap (支持丰富参数与脚本扫描)
    *   **漏洞扫描**：Nuclei (支持自定义模板)
    *   **Web测试**：SQLMap, Dirsearch
*   **MCP (Model Context Protocol) 支持**：
    *   作为 **MCP Client**，可以无缝连接任何支持 MCP 协议的外部服务或本地工具。
    *   支持管理和配置外部 MCP 服务器，实现工具能力的动态热插拔。

### 4. 📊 现代化交互与报告

*   **桌面级 UI**：基于 Wails (Go + Vue 3) 构建的跨平台桌面应用，响应迅速，界面美观，支持暗黑模式。
*   **透明化执行**：
    *   **思考过程可视化**：实时展示 AI 的"思考"（Reasoning）过程，让用户清晰看到 AI 的决策逻辑。
    *   **工具输出流**：实时查看底层工具（如 Nmap, SQLMap）的原始命令行输出，掌握每一个细节。
*   **专业报告生成**：测试结束后，自动生成包含漏洞详情、复现步骤、HTTP 请求/响应包和修复建议的标准化 Markdown 报告。

### 5. ⚙️ 高度可定制

*   **角色编辑**：支持用户修改内置角色的系统提示词（System Prompt），调整 AI 的性格和行为准则。
*   **自定义角色**：用户可以创建全新的专家角色，定义专属的 Prompt 和工具集。
*   **灵活配置**：支持自定义 OpenAI API 接口（兼容 DeepSeek 等）、工具路径、并发数、超时时间等。

## 🏗️ 技术架构

*   **Frontend**: Vue 3 + TailwindCSS + Vite (提供极致流畅的现代 Web 交互体验)
*   **Backend**: Go (Golang) (提供高性能的业务逻辑、并发处理和系统调用)
*   **Application Framework**: Wails v2 (构建轻量级跨平台桌面应用)
*   **AI Engine**: OpenAI API 兼容接口 (完美支持 DeepSeek-V3, GPT-4, Claude 等先进模型)
*   **Protocol**: Model Context Protocol (MCP) (标准化的工具与 AI 交互协议)

## 🚀 适用场景

*   **红队/渗透测试人员**：作为高效的 Copilot，辅助完成繁琐的信息搜集、端口扫描和初步漏洞验证工作，释放人力专注于高价值漏洞挖掘。
*   **安全研究员**：通过自定义 Prompt 和工具集成，快速构建特定漏洞类型的自动化挖掘流程。
*   **CTF 选手**：辅助解题，快速分析题目环境，生成解题思路。
*   **企业安全建设**：用于定期的内部资产巡检和基线扫描，及时发现暴露面。
