

# 🛡️ AI-Vanguard (先锋者)

### 智能自动化安全测试平台 | The Intelligent Automated Security Agent

## 📖 简介

**AI-Vanguard (先锋者)** 是新一代基于 **大语言模型 (LLM)** 驱动的自动化渗透测试与信息搜集平台。

不同于传统的工具调用脚本，AI-Vanguard 是一个具备**“思考”**能力的智能体 (Agent)。它基于 **ReAct (Reasoning and Acting)** 范式构建，能够模拟人类安全专家的思维过程：理解用户意图、自主制定测试计划、动态调用安全工具，并根据反馈结果实时修正行动路径。

> **核心愿景**：降低安全测试的技术门槛，同时作为专业渗透人员的高效 Copilot，由繁入简，从容应对。

---

## ✨ 核心功能亮点

### 1. 🧠 AI 驱动的智能决策核心

摒弃死板的线性脚本，AI-Vanguard 拥有动态的决策大脑：

* **🎯 自主规划**：根据目标状态实时生成测试路径。
* *示例流程*：发现 80 端口  识别 Web 服务  启动漏洞扫描  发现参数  验证注入。


* **🔄 自动纠错**：工具执行失败或参数错误时，AI 会分析错误日志（Stderr），自动修正参数并重试，或智能切换替代方案。
* **🔗 长链推理**：支持多轮对话迭代，能够处理复杂的逻辑推理任务，一步步深入渗透目标。

### 2. 🎭 多专家角色模式 (Multi-Role System)

内置多种预设专家角色，针对不同场景提供最专业的测试策略。

| 角色图标 | 角色名称 (Role) | 功能描述 & 擅长领域 |
| --- | --- | --- |
| ✨ | **智能模式 (Smart)** | **全自动托管**。智能识别用户输入与上下文，自动切换至最合适的专家角色，无需人工干预。 |
| 🛡️ | **渗透专家 (Pentester)** | **综合性测试**。覆盖从信息搜集、端口扫描到漏洞利用的全流程操作。 |
| 🔍 | **信搜专家 (Recon)** | **前期侦察**。专注于资产发现、子域名枚举、端口扫描、指纹识别及 Whois 查询。 |
| 💉 | **SQL注入专家 (SQLi)** | **数据库攻防**。专注于注入漏洞探测与验证，深度集成 SQLMap 等专业工具。 |
| 💻 | **RCE测试专家 (RCE)** | **高危漏洞挖掘**。专注于远程代码执行 (RCE) 漏洞的发现与利用验证。 |
| 🌐 | **XSS测试专家 (XSS)** | **前端安全**。专注于跨站脚本攻击的检测、Payload 构建与验证。 |
| 📂 | **文件包含专家 (LFI)** | **文件系统安全**。专注于本地/远程文件包含漏洞的探测。 |
| 🔑 | **越权测试专家 (IDOR)** | **逻辑漏洞**。专注于业务逻辑缺陷和越权访问测试。 |
| 🔌 | **MCP 模式** | **无限扩展**。作为 Model Context Protocol 客户端，连接外部服务。 |

### 3. 🛠️ 强大的工具生态与 MCP 支持

AI-Vanguard 通过集成主流安全工具和先进协议，构建了强大的能力底座：

* **📦 内置/集成工具集**
* **资产发现**：`Subfinder`, `Httpx`, `Whois`
* **网络扫描**：`Nmap` (支持 NSE 脚本与复杂参数)
* **漏洞扫描**：`Nuclei` (支持加载自定义 PoC 模板)
* **Web 深度测试**：`SQLMap`, `Dirsearch`


* **🔌 MCP (Model Context Protocol) 架构**
* **作为 MCP Client**：无缝连接任何支持 MCP 协议的外部服务或本地工具。
* **热插拔能力**：支持动态管理和配置外部 MCP 服务器，实现工具能力的无限扩展。



### 4. 📊 现代化交互与报告

* **🖥️ 桌面级 UI 体验**
基于 **Wails (Go + Vue 3)** 构建的跨平台桌面应用。界面美观，响应迅速，完美支持深色模式 (Dark Mode)。
* **👁️ 透明化执行 (White-box Execution)**
* **思考可视化**：实时展示 AI 的 `Reasoning` 过程，让用户清晰看到 AI 为什么这么做。
* **工具输出流**：实时透传底层工具（如 Nmap, SQLMap）的原始命令行输出，掌控每一个字节的细节。


* **📝 专业报告生成**
测试结束后，自动生成标准化的 **Markdown 报告**，包含：
* 漏洞详情与等级
* 完整复现步骤
* HTTP 请求/响应包截图/文本
* 专业的修复建议
![image](https://github.com/kk12-30/AI-Vanguard/blob/main/p1223.png)



### 5. ⚙️ 高度可定制化

* **角色编辑**：自由修改内置角色的 `System Prompt`，调整 AI 的性格、语气和行为准则。
* **自定义专家**：创建全新的专家角色，定义专属的 Prompt 和工具集（例如：创建一个专门针对 WordPress 的扫描专家）。
* **灵活配置**：
* **LLM 后端**：支持自定义 OpenAI 格式接口（完美兼容 DeepSeek, GPT-4o 等）。
* **运行参数**：自定义工具路径、并发线程数、超时时间等。

## 🚀 适用场景

* **🕵️‍♂️ 红队/渗透测试人员**
作为高效 Copilot，辅助完成繁琐的信息搜集和初步打点，释放精力专注于高价值漏洞挖掘。
* **🔬 安全研究员**
通过自定义 Prompt 和工具集成，快速构建特定漏洞类型（如 Log4j, Fastjson）的自动化挖掘流水线。
* **🚩 CTF 选手**
辅助解题，快速分析题目环境，生成解题思路与 Payload。
* **🏢 企业安全建设**
用于定期的内部资产巡检和基线扫描，及时发现暴露面，收敛攻击路径。
